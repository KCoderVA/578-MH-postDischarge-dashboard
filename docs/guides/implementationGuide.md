## 1. Prerequisites

*To be documented at v1.0 release. Expected prerequisites:*

- Access to your facility's VA Corporate Data Warehouse (CDW) SQL Server
- A collaborative database (e.g., `D0X_VISNxxCollab`) for stored procedure deployment
- PowerBI Desktop (Gov Cloud compatible)
- PowerBI Pro license with access to a Gov Cloud workspace
- Network access to CDW from your workstation
- Appropriate data access agreements and IRB clearance (if required)
